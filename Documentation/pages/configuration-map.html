Google Map Configuration

Step 1 : Change Google Map Api Key For Android

Specify your API Key in the application manifest android/app/src/main/AndroidManifest.xml

<manifest xmlns:android="http://schemas.android.com/apk/res/android" package="com.example">
<application android:name="com.google.android.geo.API_KEY" android:value="YOUR_KEY_HERE">

Step 2 : Change Google Map Api Key for iOS

Specify your API key in the application delegate ios/Runner/AppDelegate.swift

#include "AppDelegate.h" #include "GeneratedPluginRegistrant.h" #import "GoogleMaps/GoogleMaps.h"
@implementation AppDelegate
-(BOOL)application:(UIApplication*)application
didFinishLaunchingWithOptions:(NSDictonary*)launchOptions {
[GMSServices provideAPIKey:@"YOUR_KEY_HERE"];
[GeneratedPluginRegistrant registerWithRegistry:self];
return [super application:application didFinishLaunchingWithOptions:launchOptions];
}
